# iOS-Calculator
A GUI Calculator for Assignment 1
Made by Dov Royal 11995305
All code is contained in /Calculator/ViewController.swift
This calculator has been simulated using the iPhone 11 Pro Max

Some function of the calculator:
1. Operation buttons include +, -, /, x and = buttons
2. An operation cannot be pressed when there is nothing on the screen
3. The equals sign cannot be pressed when there is nothing on the screen
4. An operation cannot be pressed right after an operation
5. After the calculator has evaluated an operation, the result will clear upon the user pressing a new button
6. The percentage button will take the last number the user has inputted and divide it by 100 (e.g 12 becomes 0.12)
